      26                 :            : /*
      27                 :            :  * This is the Branch and Bound Coin Selection algorithm designed by Murch. It searches for an input
      28                 :            :  * set that can pay for the spending target and does not exceed the spending target by more than the
      29                 :            :  * cost of creating and spending a change output. The algorithm uses a depth-first search on a binary
      30                 :            :  * tree. In the binary tree, each node corresponds to the inclusion or the omission of a UTXO. UTXOs
      31                 :            :  * are sorted by their effective values and the tree is explored deterministically per the inclusion
      32                 :            :  * branch first. At each node, the algorithm checks whether the selection is within the target range.
      33                 :            :  * While the selection has not reached the target range, more UTXOs are included. When a selection's
      34                 :            :  * value exceeds the target range, the complete subtree deriving from this selection can be omitted.
      35                 :            :  * At that point, the last included UTXO is deselected and the corresponding omission branch explored
      36                 :            :  * instead. The search ends after the complete tree has been searched or after a limited number of tries.
      37                 :            :  *
      38                 :            :  * The search continues to search for better solutions after one solution has been found. The best
      39                 :            :  * solution is chosen by minimizing the waste metric. The waste metric is defined as the cost to
      40                 :            :  * spend the current inputs at the given fee rate minus the long term expected cost to spend the
      41                 :            :  * inputs, plus the amount by which the selection exceeds the spending target:
      42                 :            :  *

      44                 :            :  *
